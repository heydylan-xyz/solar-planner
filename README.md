# Jackery Solar Trip Planner

A concept by [heydylan.xyz](https://heydylan.xyz). Not affiliated with the Jackery brand.

An interactive 4-stage solar power planning tool. Select your devices, match a power station to your energy needs, configure your solar panel setup, and get a full system summary with pricing.

## Stack

React 18 · TypeScript · Vite · Tailwind CSS v4 · Motion

## Running locally

npm i
npm run dev
